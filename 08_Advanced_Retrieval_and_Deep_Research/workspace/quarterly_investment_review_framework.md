### Appendix B: Key Metrics Definitions

#### B.1 Performance Metrics

**Sharpe Ratio:** 
(Portfolio Return - Risk-free Rate) / Portfolio Standard Deviation
*Measures risk-adjusted return relative to a risk-free investment*

**Information Ratio:** 
(Portfolio Return - Benchmark Return) / Tracking Error
*Measures active return per unit of active risk*

**Sortino Ratio:** 
(Portfolio Return - Target Return) / Downside Deviation
*Similar to Sharpe ratio but only considers downside volatility*

**Maximum Drawdown:** 
Maximum percentage decline from peak to trough
*Measures largest single loss experienced during the period*

#### B.2 Risk Metrics

**Beta:** 
Covariance(Portfolio, Benchmark) / Variance(Benchmark)
*Measures portfolio's sensitivity to benchmark movements*

**Tracking Error:** 
Standard deviation of excess returns vs benchmark
*Measures consistency of active management*

**Value at Risk (VaR):** 
Potential loss at specified confidence level over specified time period
*Estimates worst-case loss under normal market conditions*

**Expected Shortfall:** 
Average loss beyond the VaR threshold
*Estimates average loss in tail scenarios*

#### B.3 Alternative Investment Metrics

**Internal Rate of Return (IRR):** 
Discount rate that makes NPV of cash flows equal to zero
*Measures annualized effective compounded return rate*

**Multiple of Invested Capital (MOIC):** 
Total Value / Total Invested Capital
*Measures total return multiple on invested capital*

**Distributed to Paid-In (DPI):** 
Cumulative Distributions / Cumulative Contributions
*Measures cash-on-cash return from distributions*

**Residual Value to Paid-In (RVPI):** 
Net Asset Value / Cumulative Contributions
*Measures unrealized value relative to invested capital*
